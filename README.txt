tiny cli for finance management

Запуск:
  python finansic.py

Данные хранятся в файле finance.json в текущей рабочей директории

# Главное меню:
[0] add — добавить запись (дата dd.mm.yyyy, пусто = сегодня; категория; сумма; тип income(i) / expense(e))
[1] view — просмотр записей: все(a) / только доходы(i) / только расходы(e)
[2] edit — редактировать запись по номеру в полном списке; пустые поля оставляют старое значение
[3] delete — удалить запись по номеру (подтверждение словом «да»)
[4] balance — установить итоговый баланс (новое число полностью заменяет предыдущий итог; сами записи не удаляются)
[5] analytics — аналитика (подменю ниже)
[6] exit — выход из программы
[7] help — показать список команд снова

Команды можно вводить номером или slug (add, view, edit, delete, balance, analytics, exit).

# Аналитика ([5] analytics):
[0] Доходы и расходы за всё время
[1] Доходы и расходы за прошлый календарный месяц
[2] Доходы и расходы за выбранный период (две даты dd.mm.yyyy)
[3] Список категорий (по всем записям)
[4] Траты по категориям за всё время (сортировка по убыванию суммы)
[5] Траты по категориям за прошлый календарный месяц
[6] Траты по категориям за выбранный период
[7] Полный отчёт по месяцам: сводка, категории и траты по категориям для каждого месяца
